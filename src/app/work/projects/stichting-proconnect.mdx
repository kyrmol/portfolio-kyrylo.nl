## Overview

Built and deployed the public website for Stichting ProConnect, a Dutch foundation. The site is built with Next.js and hosted on AWS Amplify, providing a fast and maintainable web presence for the organization.

## Tech Stack

- **Next.js** — React framework for server-side rendering and static generation
- **AWS Amplify** — hosting, CI/CD and deployment pipeline straight from Git

## Key Details

- Configured AWS Amplify for automatic deployments on push
- Optimized for performance and accessibility
- Live at [stichtingproconnect.nl](https://stichtingproconnect.nl)
